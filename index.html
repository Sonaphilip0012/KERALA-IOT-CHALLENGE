Kerala IoT Challenge

Foxlab Makerspace in association with GTech - Group of Technology Companies in Kerala is launching their prestigious program “Kerala IoT Challenge 2021”, with a vision to mould 100 IoT experts in Kerala, hosting on the MuLearn platform. Kerala IoT Challenge is a program designed in 4 levels followed by a hackathon to identify and train quality industry leaders in the IoT domain, while any novice learner can start with layer 1 and others can enter laterally to the desired layer after an evaluation.

This is a project website developed to feature all of my experiments, assignments, projects, and the insights I was able to gain from doing them.

About The Challenge

"Kerala IoT Challenge" is a 6 month long program were we will learn to do IoT projects through 4 different levels and it is followed by a hackathon.

Level 01: Basic Electronics & Arduino

Basic Electronics, Sensors, Arduino Programming and a Live Project

Level 02: Introduction to IoT

ESP32 Programming, Home Automation Projects and a Live Project

Level 03: IoT Intermediate

Server Side Programming, Communication Protocols and Advanced Project

Level 04: IoT Advanced

IoT Product Design & Fabrication, Cloud and IIoT
